## Custom NVim configuration of Josean Martinez NVim setup guide

Plugins included:

- lazy.nvim plugin manager
- vim-tmux-navigator
- tokyonight-light colorscheme
- file explorer
- which-key (keymapping legend)
- fuzzy finder
- greeter
- auto sessions saver
- bufferline
- statusline
- dressing.nvim
- splits maximizer
- treesitter
- indentation
- autocompletion
- autopairs
- comments
- todo comments
- substitution
- surround plugin
- lsp (attached by mason package manager)
- trouble.nvim
- formatting
- linting
- git integration
